## 🎯 INTEGRATING WITH WARRIOR PLUS

### **How to Add Buyers to GetResponse Automatically:**

**Option 1: Warrior Plus Webhook (Recommended)**

1. Go to Warrior Plus → **Product Settings**
2. Find **"Integration Settings"**
3. Add **GetResponse API Key**:
   - Get from GetResponse: **Menu** → **Integrations & API** → **API**
   - Copy API Key
   - Paste into Warrior Plus
4. Select list: `Faith Funnels AI - Buyers`
5. Save settings

**Now:** Every purchase automatically adds buyer to GetResponse and triggers Welcome Sequence!

---

**Option 2: Manual CSV Import (Backup Method)**

1. Export buyers from Warrior Plus (CSV)
2. Go to GetResponse → **Contacts** → **Import**
3. Upload CSV
4. Map fields (Email, First Name, Last Name)
5. Select list: `Faith Funnels AI - Buyers`
6. Import

## 📊 GETRESPONSE BEST PRACTICES

### **Deliverability Tips:**

1. **Use Plain Text Emails** - Higher deliverability than HTML
2. **Avoid Spam Words** - Don't overuse: FREE, BUY NOW, ACT NOW, etc.
3. **Include Physical Address** - GetResponse adds this automatically
4. **Test Before Sending** - Use "Send Test" feature
5. **Check Spam Score** - GetResponse shows spam score before sending

### **Personalization:**

- `{{FNAME}}` - First name
- `{{LNAME}}` - Last name
- `{{EMAIL}}` - Email address

### **Tracking & Analytics:**

1. Go to **Reports** → **Newsletters**
2. Monitor:
   - Open rate (aim for 20-30%)
   - Click rate (aim for 2-5%)
   - Unsubscribe rate (keep below 0.5%)

### **A/B Testing:**

1. Create newsletter
2. Click **"A/B Test"**
3. Test:
   - Subject lines
   - Send times
   - Email content
4. GetResponse sends winning version automatically
